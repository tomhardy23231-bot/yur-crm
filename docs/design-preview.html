Как это работает

  • Шапка — indigo gradient (#4F46E5 → #7C3AED).
  • Аватар на фоне градиента — белая обводка + полупрозрачный pill для actions.
  • Секции «Деталі / Контакти» сворачиваются (chevron справа).
  • Inputs без border — на surface-muted фоне. Border + focus-ring только на active.
  • Чип компании — primary-subtle с цветным dot + бренд-тег («Lead» в info-blue).
  • Иконки источников — в брендовых цветах (Gmail красный, Telegram синий, WhatsApp зелёный, Viber фиолетовый).
