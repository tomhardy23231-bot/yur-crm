# Глоссарий перевода RU → UK (Юр CRM)

Единый источник согласованности терминов для двуязычного интерфейса.
Русский — эталон (текущий UI), украинский — целевой перевод. Юридический
домен, Украина. Использовать ИМЕННО эти варианты во всех экранах.

## Правила

- Множественное число (1 дело / 2 дела / 5 дел) — через helper `plural(forms, n)`
  с формами `{ one, few, many }` (Intl.PluralRules для uk).
- Дни/штуки: 1 день/3 дні/5 днів; 1 справа/2 справи/5 справ; 1 завдання/3 завдання/5 завдань.
- Апостроф в украинском — символ `’` (U+2019), НЕ прямая кавычка (ломает JS-строку):
  обов’язково, пов’язані, прізвище (без апострофа), сім’я.
- «ё» в русском оставляем как в оригинале (не трогаем русский — он эталон).
- Email / Legal CRM / ЮрКейс — не переводим.
- Деньги/числа форматируются кодом (Intl) — переводим только подписи, не значения.
